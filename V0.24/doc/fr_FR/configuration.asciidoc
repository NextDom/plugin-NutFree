Nous allons maintenant paramtrer un quipement. Pour se faire, cliquer sur *Plugins  / Monitoring / Nut_free*

Puis cliquer sur le bouton en haut  gauche *Ajouter*

image::../images/Nut_free_1.png[align="center"]


Puis saisir le nom de l'quipement (ex. Onduleur_Nas) 

image::../images/Nut_free_2.png[align="center"]

Puis dfinir :

- Objet parent
- Catgorie (optionnelle)
- Activer ( cocher, sinon lquipement ne sera pas utilisable)
- Visible (optionel si vous ne dsirez pas le rendre visible sur le Dashboard)

image::../images/Nut_free_3.png[align="center"]



Adresse IP:: IP du serveur NUT distant
Nom de la configuration UPS:: Nom de l'onduleur a surveill vu par NUT ( Rsultat de la commande upsc -l).
							 - Si il y a 1 seul onduleur sur le serveur NUT et que la connexion ne se fait pas par ssh, laisser le champs vide.
							  Le plugin trouvera seul le nom.
							 - Si il y a plusieurs onduleur, il faudra connaitre le nom et le renseigner.
